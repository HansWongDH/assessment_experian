## How to Run

### Using Makefile

1. Open a terminal.
2. Run the following command:

    ```bash
    make
    ```

### Using Docker Compose

1. Open a terminal.
2. Run the following command:

    ```bash
    docker-compose up
    ```

### Running Frontend

1. Navigate to the "frontend" directory.
2. Run the following command:

    ```
    yarn start dev
    ```
